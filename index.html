Add Tasks
  • To add tasks to your task list, begin by typing in the task input and click the button or press the Enter key.
Remove Tasks
  • To remove tasks from your task list, click on the button.
  • If the selected task is important the button will be disabled (See Task Priority for more).
Completed Tasks
  • If you've completed a task and want to mark it down as done, click the button.
  • When a task is completed, the task name will go green. This indicates a completed task.
Task Priority
  • If your task is important you can click on the button.
  • When clicked, the button will be disabled. This stops your task from being removed from your task list by accident.
  • Important tasks name will go red. This indicates an important task.
  • You can also reorder your tasks by using the and buttons.
